¿Quiénes Somos?

El IoT y la necesidad de las personas en el manejo de información remota, control de elementos y supervisión de sus diferentes propiedades ha llevado al desarrollo de HappyPlant.


Somos un equipo de estudiantes del politécnico grancolombiano sede Bogotá que brindando una solución al control y supervisión remota de las plantas, ayuda a las personas a tener un control de estas y mantenerlas saludables.
